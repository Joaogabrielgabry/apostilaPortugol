Aula 4 – Algoritmo

Algoritmo é uma sequência organizada de passos para resolver um problema ou executar uma tarefa. Deve ter início, ordem lógica e fim.

No cotidiano, utilizamos algoritmos o tempo todo, mesmo sem perceber. Receitas, manuais e tutoriais são exemplos práticos.

Exemplo (vida real)

Para beber água:

  1. Pegar a garrafa
  2. Abrir a tampa
  3. Levar à boca
  4. Beber a água

A ordem é fundamental. Se inverter os passos, o resultado não funciona.

Algoritmo na Programação

Na programação, a lógica é idêntica. O computador executa exatamente os passos definidos.

Exemplos:
  • Passos para calcular média
  • Verificar número par ou ímpar
  • Criar sistema de login

Se faltar um passo ou a ordem estiver errada, o programa não funciona corretamente.

Por que aprender algoritmos?

  • Pensamento organizado
  • Divisão de problemas em partes menores
  • Construção de soluções estruturadas
  • Compreensão da lógica do computador

Antes do código, vem a ideia. Primeiro o algoritmo. Depois a implementação.
