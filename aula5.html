Aula 5 – Estrutura Básica de um Programa em Portugol

Todo programa em Portugol segue uma estrutura padrão. Essa organização mantém o código claro e fácil de manter.

Exemplo completo

algoritmo "Exemplo"

var
   nome: caractere

inicio
   escreva("Digite seu nome: ")
   leia(nome)
   escreva("Olá, ", nome)

fimalgoritmo

Explicando cada parte

algoritmo

Define o nome do programa. Serve para identificar sua finalidade. 

algoritmo "Cadastro"

var

Área destinada à declaração das variáveis. Aqui o programa reserva espaço na memória. 

var
   idade: inteiro
   nome: caractere

inicio

Marca o início da execução do programa. Todas as instruções executáveis ficam aqui.

fimalgoritmo

Indica o encerramento do programa. Após isso, não há mais execução.

Pontos importantes

  • Todo programa começa com algoritmo
  • Variáveis ficam dentro de var
  • Execução ocorre entre inicio e fimalgoritmo
  • Organização reduz erros

Por que isso é importante?

  • Evita confusão
  • Facilita manutenção
  • Melhora legibilidade

Criar padrão desde o início desenvolve disciplina técnica. Esse hábito acelera a transição para linguagens profissionais.
