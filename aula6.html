Aula 6 – Variáveis

Uma variável é um espaço reservado na memória para armazenar dados que podem mudar durante a execução.

Pode ser comparada a uma caixa com nome, onde guardamos valores para usar depois.

Sem variáveis, não existe processamento. Elas são a infraestrutura básica de qualquer algoritmo.

Para que servem?

  • Guardar dados do usuário
  • Armazenar resultados
  • Controlar decisões
  • Repetir processos
  • Organizar informações

Exemplo: armazenar a idade digitada pelo usuário.

Declaração de Variáveis

var
   idade: inteiro
   salario: real

idade e salario são nomes. inteiro e real são os tipos.

Tipos Básicos

Inteiro

var
   quantidade: inteiro

Real

var
   altura: real

Caractere

var
   nome: caractere

Lógico

var
   aprovado: logico

Valores possíveis: verdadeiro ou falso

Atribuição de Valores

idade <- 20
salario <- 1500.50
nome <- "Carlos"
aprovado <- verdadeiro

Leitura de Dados

escreva("Digite sua idade: ")
leia(idade)

Uso em Cálculos

var
   a, b, soma: inteiro

a <- 5
b <- 3
soma <- a + b

escreva("Resultado: ", soma)

Regras para Nomes

  • Não começar com número
  • Não usar espaços
  • Não usar acentos
  • Não usar símbolos especiais
  • Ser claro e objetivo

✔ idadeAluno
✔ salario
✔ mediaFinal

❌ 2valor
❌ nome aluno
❌ salário

Boas Práticas

  • Use nomes descritivos
  • Evite x, y, a
  • Declare no início
  • Escolha o tipo correto

Exemplo Completo

algoritmo "ExemploVariaveis"

var
   nome: caractere
   idade: inteiro
   altura: real

inicio
   escreva("Digite seu nome: ")
   leia(nome)

   escreva("Digite sua idade: ")
   leia(idade)

   escreva("Digite sua altura: ")
   leia(altura)

   escreva("Nome: ", nome)
   escreva("Idade: ", idade)
   escreva("Altura: ", altura)

fimalgoritmo

Conclusão

Variáveis tornam o algoritmo dinâmico. Permitem armazenar, modificar e utilizar informações.

Sem variáveis, o programa seria fixo, sem flexibilidade ou tomada de decisão.
