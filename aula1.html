Aula 1 – Apresentação

Esta apostila foi criada para ajudar você a dar os primeiros passos no mundo da programação, de forma simples, organizada e sem complicação. Aqui, você vai aprender a pensar como um programador, mesmo antes de usar linguagens famosas como Python, Java ou C.

A proposta é começar pelo básico e evoluir de forma progressiva. Primeiro, você vai entender como resolver problemas usando lógica, passo a passo. Depois, vai aprender a transformar esse raciocínio em algoritmos e programas usando Portugol, uma linguagem feita especialmente para iniciantes.

Foco Principal

  • Pensar
  • Organizar ideias
  • Resolver problemas
  • Criar soluções com lógica

Aqui, errar faz parte do aprendizado. Cada exercício é uma oportunidade de evoluir e entender como o computador processa informações.

Ao finalizar esta apostila, você estará preparado para aprender linguagens como Python, Java ou C com muito mais segurança, porque a base lógica será sólida.
